Detailed Description

Class responsible for the energy calibration of the charge spectra. Calibration is performed using peaks defined by the user in the configuration file (see DDCalib class reference) and stored in fPeaks vector. Sum of Gaussian function and polynomial of the 2nd order is fitted to the defined peaks. Subsequently peak ID energy is plotted as a function of their mean values and linear fit is performed. Parameters of that fit are the parameters of the energy calibration for regular measurements.
